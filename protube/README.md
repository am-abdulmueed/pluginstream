## Features
 * <img src='https://raw.githubusercontent.com/am-abdulmueed/protube/main/.github/img/gemini-logo-13486188-10900314-unscreen-ezgif.com-crop.gif' height=15 width=15 > Google Gemini
   * Summarise Vidoes
   * Customisable prompts and models

 * Shorts Downloader
